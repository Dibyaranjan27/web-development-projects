## ğŸš€ Getting Started

### **Quick Start**

1. **Clone the Repository**
   ```sh
   git clone https://github.com/Dibyaranjan27/lumen.git
   cd lumen
   ```

2. **Open in Browser**
   Simply open the `index.html` file in your web browser:
   ```sh
   # On Windows (PowerShell)
   start index.html
   
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   ```

3. **Optional: Use a Local Server**
   For better performance and to avoid CORS issues, serve the files using a local server:
   ```sh
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js with http-server
   npx http-server
   ```
   Then visit `http://localhost:8000` (or the port shown) in your browser.
